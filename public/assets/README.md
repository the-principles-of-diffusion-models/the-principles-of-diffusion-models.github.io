# Assets Directory

This directory is for storing images, figures, GIFs, and other media assets used in the blog posts and pages.

## Usage

Place your image files here and reference them in your components using:

```jsx
<img src="/assets/your-image.png" alt="Description" />
```

## File Organization

- Keep filenames descriptive and lowercase
- Use hyphens to separate words (e.g., `forward-process-diagram.png`)
- Organize by type if needed (e.g., `diagrams/`, `photos/`)

## Supported Formats

- PNG (recommended for diagrams and screenshots)
- JPG/JPEG (recommended for photos)
- GIF (for animations)
- SVG (for vector graphics)
- WebP (for optimized web images)
